AutoDetect Template Filler

Vercel-hosted • Gemini Vision • Cloud Storage • Multi-Template Support

Workflow:
  1. Upload a template image (stored in Vercel Blob cloud)
  2. Click Auto-Detect with Gemini for intelligent field recognition
  3. Tick checkboxes to select fields you want
  4. Click Apply Selected — they become spreadsheet columns
  5. Drag boxes to move. Drag green handles to resize
  6. Image fields accept image uploads in spreadsheet cells
  7. Fill data and Render All
  8. Save Template to cloud — access from anywhere, switch between templates
